Get connected in a few clicks

Config-driven — no code edits. Bring your own Supabase + GitHub OAuth.

1 · Create Supabase

Spin up a free project, then apply the two SQL migrations in supabase/migrations.

2 · Enable GitHub

Auth → Providers → GitHub: paste your GitHub OAuth App's client id/secret.

3 · Connect in-app

Settings → Backend connection: paste the project URL + anon key. Stored locally, auto-reconnects.

4 · Continue with GitHub

One tap to sign in. The worker reads keys from config — verify with /config/status.

Where it's at

This is an early, honest preview. The Flutter app ships the full design system, Reels UX, a Startup Seed page, an AI-providers config screen, and a click-through Supabase connect + GitHub sign-in flow. The feed still renders mock data until live persistence is wired. The APK is a debug-signed sideload preview.
